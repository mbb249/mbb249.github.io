QBERT

Of all my projects, this one was certainly my favorite. This was my final project for CS3110 (Functional Programming), and I completed this project with three other students. It took us about 3 months to come up with ideas, plan out our code, and then ultimately write all the code and tests cases. In the end, this project amassed over 1600 LOC. I mainly focused on developing the Player module and the Itemshop/Cutscenes. All of the code was written in OCaml, and we mainly utilized the Graphics module to create our game.
